# Шаблон отчета по практике

1. Титульный лист (специальность 06130100, модуль ПМ01)
2. Содержание
3. Введение (цель, задачи, стек)
4. Основная часть
   - Архитектура: ERD, схема связей 1:1, 1:N, M:N
   - Модель данных: таблицы, ключи, уникальность, индексы
   - SQL-запросы: примеры простых/сложных (см. `docs/sql_examples.sql`)
   - Автоматизация: скрипты миграций, бэкапов/восстановления, команды ролей
   - Безопасность: роли/права, политика бэкапов, работа с секретами
   - Тестирование: перечень тестов, результаты прогонов
   - Производительность: индексы, агрегации, оптимизация отчетов
   - Интерфейс: описать страницы, добавить скриншоты UI и базы данных
5. Заключение (выводы, что улучшить)
6. Список литературы/источников
7. Приложения (скриншоты, дампы, дополнительные отчеты)
