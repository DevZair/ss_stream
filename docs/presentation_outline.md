# Конспект презентации для защиты

1. Титульный слайд: тема, ФИО, модуль ПМ01
2. Цель и задачи проекта (управление складом, учет движений и продаж)
3. Архитектура БД: ERD, ключи, связи 1:1/1:N/M:N
4. Основные сущности и уникальные идентификаторы (`Warehouse.code`, уникальные категории, товары)
5. Демонстрация функций: приход, перемещение, продажа, отчеты, экспорт CSV
6. Автоматизация: миграции, бэкап/restore скрипты, команды создания суперпользователя и ролей
7. Безопасность: группы прав (Администратор/Менеджер/Оператор), политика бэкапов
8. Тестирование и качество данных: проверки на недостачу, запрет редактирования операций, smoke по отчетам
9. Производительность: индексы по датам, агрегации, оптимизация запросов
10. Итоги и ответы на вопросы
